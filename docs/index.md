# Data Structures and Algorithms Repository

A comprehensive collection of DSA implementations in Python, Java, and C++.

## 📁 Repository Structure

### Python Implementations 🐍

All Python code is organized in the `python/` folder:

- **Interview Preparation** - LeetCode solutions and coding interview problems
- **Data Structures in Python** - Stacks, Queues, Linked Lists, Hash Maps, Recursion
- **Arrays** - Array manipulation and intersection problems
- **String Algorithms** - String operations and palindrome problems
- **LeetCode Solutions** - Categorized by difficulty and topic
- **CodeChef Solutions** - Competitive programming problems

### Java & C++ Implementations ☕🔧

Combined implementations in the `javaandcpp/` folder:

#### Java (`javaandcpp/java/`)
- Fibonacci sequence calculations
- GCD and LCM algorithms
- Mean calculations
- Palindrome checking
- Pattern printing

#### C++ (`javaandcpp/cpp/`)
- Binary Search algorithms
- Binary Tree operations
- Linked List implementations
- Graph algorithms
- Recursion problem solutions
- Sorting algorithms
- String operations

## 📚 Getting Started

1. Navigate to the `python/` folder for Python implementations
2. Navigate to the `javaandcpp/` folder for Java and C++ code
3. Each subfolder contains organized solutions by topic
